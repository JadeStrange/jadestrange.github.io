# git命令

## 命令行类似图形化显示提交记录

```sh
git log --oneline --all --graph
```

![1](/images/2020-12-17-git-git/1.png)

## 回退

```sh
git checkout -b new_name  # 备份
git log --oneline --all --graph  # 回到原先的分支，通过git log查看历史commit。
git reset --hard <commit id> (不含<>号)
```
